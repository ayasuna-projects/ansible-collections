This collection contains utility roles & plugins.

### Roles

| Role FQN                  | Description                                                                       |
|---------------------------|-----------------------------------------------------------------------------------|
| ayasuna.utilities.acme    | Defines tasks that can be used to issue Let's Encrypt certificates                |
| ayasuna.utilities.host    | Contains tasks which can be used to configure different aspects of a (linux) host |
| ayasuna.utilities.ingot   | Defines tasks which can be used to build and deploy ingots                        |
| ayasuna.utilities.udevadm | This role can be used to perform a few `udevadm` operations                       |

### Plugins

| Plugin FQN                          | Description                                                                                |
|-------------------------------------|--------------------------------------------------------------------------------------------|
| ayasuna.utilities.prepare_arguments | Can be used to prepare (validate & convert) arguments based on an extended `argument_spec` |
