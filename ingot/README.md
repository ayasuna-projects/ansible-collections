This collection contains roles that can be used as argument for the `ayasuna__utilities__ingot__inputs__build_ansible_role_fqn`
parameter of the `ayasuna.utilities.ingot` role to build ingots.

### Roles

| Role FQN             | Description                                                                                             |
|----------------------|---------------------------------------------------------------------------------------------------------|
| ayasuna.ingot.k8s    | Can be used to build a Kubernetes ingot. <br/>The ingot can then be used to setup a Kubernetes cluster. |
| ayasuna.ingot.rescue | Can be used to build a rescue ingot.                                                                    |
